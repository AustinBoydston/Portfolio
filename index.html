Maze Generator

This project was for my students begginer programming final project. They were to make a maze game that generates a maze then allows the player to navigate throught it with the arrow keys until they reach the exit square. Below is a clip of my solution running. If you would like to download this and try it for yourself you can do so at This Link.The biggest lesson I took away from this project was how to integrate the pygame libray for video game development and how to design a flexible maze generation algorithm.

Minecraft Stronghold Finder

This project was a fun challenge to build a program that can calculate the location of the stronghold in minecraft using the method of triangulation. I built it out in python quickly and wanted to explore python-web integrations and stumbled upon the Brython project. I built out a static site on github pages and inserted my python code with brython as a backend. You can try this project outhere. I am very pleased with the result and will take Brython with me as a tool I can use anytime it is more cost effective to run rather than a complete rewrite in Javascript.

Asteroids Like Space Game

This is a game I developed using game maker studio engine that you can download here(The password is spacegame). I have used the experience from this game to teach students how to create similar 2d games in godot. All the art and music was my own work and I gained an appreciation for working in an engine to build games. What I learned most here was how to approach game frame rendering, collision, and timing in a live simulation. Godot is a very similar engine that's free and my students have loved working with it.

The controls for the game are: w, a, s, d to move, spacebar to shoot, and m for mines.
